新Mac セットアップ

2段階運用 — 両ステップとも冪等。Safariでこのページを開きつつ、Terminalで上から順にCopy→⌘Vで実行する。

Stage 1 — 物理画面で（SSH + ホスト名）

新Mac の Terminal で実行。途中で sudo パスワードを聞かれるので、ログインパスワードを入力。

curl -fsSL rojilab.github.io/dotfiles/setup-stage1 -o setup-stage1
bash setup-stage1

SSH (Remote Login) を有効化し、ホスト名を hoshilab に設定する。完了したら別マシンから ssh <username>@hoshilab.local で入れる。

Stage 2 — 別マシンから ssh で入って

Stage 1 完了後、別マシン（td2026 等）から ssh hoshilab で入り、新Mac側で下記を実行。

curl -fsSL rojilab.github.io/dotfiles/setup -o setup
bash setup

Homebrew, Brewfile, macOS defaults, symlink 等のフル setup。Dropbox cask の GUI ログインだけは物理画面に戻る必要がある（同期完了後、もう一度 bash setup で symlink 張りが走る）。

オプション

bash setup -a   # + admin packages (sudo)
bash setup -x   # + extended packages
bash setup -f   # + Thunderbird/Stylus checks

新Mac セットアップ

3ステップ。Step 1 と 2 は新Mac の前で。Step 3 以降は別マシンから ssh 越しで進められる。

1リモートログインを ON にする— 新Macの GUI

新Mac で システム設定 を開き、リモートログインを有効化する。

  1. メニューバー左上の Appleマーク → システム設定
  2. 左のサイドバーで 一般 → 共有
  3. リモートログイン のトグルを ON
  4. 「アクセス」は すべてのユーザ または 管理者のみ でOK
なぜ macOS Tahoe 以降、ターミナルから systemsetup -setremotelogin on を叩いても TCC/プライバシー要件で反映されない場合があるため、GUI で先に有効化する。

2ホスト名を hoshilab に— 新Macのターミナル

新Mac で ターミナル を開き（⌘ Space → 「ターミナル」）、下の Copy ボタンを押してターミナルで ⌘V + Enter。途中で sudo パスワードを聞かれるのでログインパスワードを入力。

curl -fsSL rojilab.github.io/dotfiles/setup-stage1 -o setup-stage1
bash setup-stage1

実行後、最後に表示される このマシンの IP アドレス をメモする（次のステップで別マシンから接続するため）。

3別マシンから ssh で入って setup— td2026 など別マシン

td2026 等の別マシンのターミナルで以下を実行。<NEW_IP> を Step 2 でメモした新MacのIPに置き換える。

3a. 公開鍵を新Macに送る（初回のみ。新Mac のパスワードを入力）

ssh-copy-id -i ~/.ssh/id_ed25519_eto.pub lab@<NEW_IP>

3b. ssh で新Macに入る

ssh lab@<NEW_IP>

3c. 新Mac内で setup を実行（Homebrew, Brewfile, macOS defaults などのフルセットアップ）

curl -fsSL rojilab.github.io/dotfiles/setup -o setup
bash setup
途中で物理画面に戻る Dropbox cask のインストール後、GUI でDropboxにログインが必要。同期完了後、もう一度 bash setup を実行すると symlink 張りが走る。

オプション（setup フラグ）

bash setup -a   # + admin packages (sudo)
bash setup -x   # + extended packages
bash setup -f   # + Thunderbird/Stylus checks
